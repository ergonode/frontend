# LinkButton

`LinkButton` is a default link component. It may have any icon component inside `prepend` and `append` slots.

```
    import LinkButton from '@UI/components/LinkButton/LinkButton'
```

<Preview>
    <Story name="basic">
        {{
            components: { LinkButton },
            template: `<LinkButton title="Link" />`,
        }}
    </Story>
</Preview>
